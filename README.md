# INST326Project
Desktop Widget 
## Description
The Study Companion Desktop Widget is a Python-based application designed to improve productivity by combining a Pomodoro timer with interactive mini-games. The application helps users stay focused during study sessions while providing engaging breaks to reduce burnout.

## Features
- Pomodoro timer with customizable work and break intervals
- Visual countdown display
- Mini-games available during breaks (Tic-Tac-Toe, Flappy Bird, Rock-Paper-Scissors)
- Simple graphical user interface (GUI)

## Project Structure
